# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this repository, please report it
responsibly:

1. **Do NOT** open a public issue
2. Email **security@vectrade.io** with details
3. Include steps to reproduce if possible

We will acknowledge receipt within 48 hours and provide a fix timeline within 7
business days.

## Scope

This repository contains example code and templates. Security issues in the
underlying VecTrade SDKs should be reported to the respective SDK repositories.
